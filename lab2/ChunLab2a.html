Table:a set of data organized into rows and columns

A table is created by using the table heading and closing tags to tell html that we are creating a table. We then use tr(table rows) as tabs to create a new row for the table,th(table headers) to create a header on the table, td(table descriptions) for the content in the table,thread to specify the head of the table for modifcation, tfoot for specifying the foot, and caption to make a title for the table.

The difference between colspan and rowspan is that colspan combines groups of columns together while rowspan combines groups of rows together.

The difference between cell spacing and cell padding is that cell spacing defines the space between cells and cell padding defines the spaces between the edges of cells and the content inside the cells.

to define table borders you have to use the border property and specify what elements are surrounded by the border.

to control border width you have to use the border width property and specify what value it has.
